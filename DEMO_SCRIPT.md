## 0:00–0:12 — The problem

“Every AI agent says it is an expert. Today, those claims are just profile text. ProofMarket turns
claims into evidence.”

Show the agent marketplace and SENTINEL-9's claimed Solidity capability.

## 0:12–0:28 — The sealed trial

Open the live arena.

“The challenge is committed before the agent sees it. This contract contains two exploitable bugs,
one lower-severity weakness, and one deliberate false-positive trap.”

Press **Run live capability trial**.

## 0:28–0:52 — Independent evaluation

Narrate the execution log:

“The agent works inside an observable sandbox. Its findings go to an independent jury through 0G
Compute, scored against a rubric the agent never saw.”

Reveal the source briefly as the verdict arrives.

## 0:52–1:10 — Reproducible evidence

Show the verdict and findings.

“The complete challenge, response, rubric, model and verdict are bundled and anchored to 0G Storage.
The evidence root makes any later edit detectable.”

## 1:10–1:25 — Portable Passport

Scroll to the Capability Passport.

“ProofMarket issues a non-transferable credential on 0G Mainnet. Any marketplace can require a
fresh score above its own threshold before trusting this agent with work or funds.”

## 1:25–1:30 — Close

“ProofMarket: hire the proof, not the pitch.”

End on the 0G Compute, Storage and Chain proof stack.
